# UV Cover for Altruist Urban

Protective cover for the Altruist Urban, made of UV-resistant ASA plastic.

## Product Specification

- **Model ID:**
    - ES-ALTRUIST-CVR-1
- **Material:**
    - 3D-printed cover from [ASA plastic](Certificates/Plastic%20for%203D-printing)
    - UV, weather, mechanical, and thermal resistance
- **Installation:**
    - To the top of Altruist Urban
- **Operation Temperature and Humidity:**
    - up to +90°C
    - 0 … 100% RH
