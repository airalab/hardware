# Altruist Urban — 3D Enclosure Parts (STL)

STL files for 3D-printing the Altruist Urban enclosure. Printed in PLA plastic.

## Parts

| File | Description | Qty |
|------|-------------|-----|
| Antenna.stl | Antenna for the enclosure* | 1 |
| Bottom_Part.stl | Bottom part of the enclosure | 1 |
| Central_Part.stl | Central part of the enclosure | 1 |
| Emotion_Part_Deadly.stl | "Deadly" emotion face panel** | 1 |
| Emotion_Part_Enjoy.stl | "Enjoy" emotion face panel** | 1 |
| Emotion_Part_Minimal.stl | "Minimal" emotion face panel** | 1 |
| Emotion_Part_No_Face.stl | Blank face panel** | 1 |
| Emotion_Part_Smile.stl | "Smile" emotion face panel** | 1 |
| Bbolt_M6.stl | M6 fastening bolt | 2 |
| Pressure_plate.stl | Pressure plate for the sensors board | 1 |
| SDS_Board_Locking_Clip.stl | Locking clip for the SDS011 board | 3 |
| SDS_Board_Spacer_Rail.stl | Spacer rail for the SDS011 board | 6 |
| Sensors_Board.stl | Sensors board mount | 1 |
| Wall_Mount.stl | Wall mount bracket | 1 |

\* Antenna is not needed when using Emotion_Part_Minimal.
\** Only one Emotion part needs to be printed (choose one).

## Printing

- **Material:** PLA
- **Recommended layer height:** 0.2 mm
- **No supports required** for most parts
