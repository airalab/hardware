# Altruist Urban — 3D Enclosure Parts (3MF)

3MF files for 3D-printing the Altruist Urban enclosure on Bambu Lab printers. Each file contains all parts arranged on a single plate, printed in PLA plastic. Only one file needs to be printed — choose the desired emotion variant.

## Files

| File | Description | Qty |
|------|-------------|-----|
| Altruist_Urban_Emotion_Deadly.3mf | Full enclosure set with "Deadly" emotion face panel | 1 |
| Altruist_Urban_Emotion_Enjoy.3mf | Full enclosure set with "Enjoy" emotion face panel | 1 |
| Altruist_Urban_Minimal.3mf | Full enclosure set with minimal face panel (no antenna) | 1 |
| Altruist_Urban_No_Face.3mf | Full enclosure set with blank face panel | 1 |
| Altruist_Urban_Emotion_Smile.3mf | Full enclosure set with "Smile" emotion face panel | 1 |

## Printing

- **Material:** PLA
- The first plate in each file uses a textured build plate; subsequent plates do not.
- **Recommended layer height:** 0.2 mm
- **No supports required** for most parts
