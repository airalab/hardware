FreeCAD source files for the Altruist Urban enclosure.

## Files

| File | Description |
|------|-------------|
| Altruist_Urban.FCStd | FreeCAD project containing main enclosure parts (central part, bottom part, emotion panels, antenna) |
| Altruist_Wall_Mount.FCStd | FreeCAD project for the wall mount bracket and bolt |
| Sensors_Board.FCStd | FreeCAD project for the sensors board, spacer rail, locking clip, and pressure plate |
