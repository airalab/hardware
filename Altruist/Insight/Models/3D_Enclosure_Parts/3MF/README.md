# Altruist Insight — 3D Enclosure Parts (3MF)

3MF file for 3D-printing the Altruist Insight enclosure on Bambu Lab printers. Contains all parts arranged on plates, printed in PLA plastic.

## Files

| File | Description | Qty |
|------|-------------|-----|
| Altruist_Insight.3mf | Full enclosure set with all parts | 1 |

## Printing

- **Material:** PLA (for indoor use, 0–40°C)
- The first plate uses a textured build plate; the second plate uses a smooth build plate.
- **Recommended layer height:** 0.2 mm
- **No supports required** for most parts
