## Hardware for Cyberpunks

This repository describes Hardware for Cyberpunks — a line of devices featuring fully open-source firmware and open hardware built on the open architectures. The repository pins open access final versions of all PCB models, schematics and housing models for 3D printing.

### Structure for Projects

```plaintext
└── Name/                   # Name project/device
    ...
    ├── BOM/                # Files related to bills of materials
    ├── Certificates/       # Files related to certification of devices, their components and materials
    ├── Docs/               # Technical documentation, datasheets
    ├── Img/                # Related images
    ├── Models/             # 3D models (components, housing, etc.)
    ├── PCB/                # Files with printed circuit board schemes
    ├── Project/            # Files for EDA projects
    ├── Schematic/          # Schematic diagrams of devices
    └── Typography/         # Files with products for printing (e.g. for printshops)
```
